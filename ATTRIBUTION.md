# Asset Attribution

We track every third-party asset here, including its source and license —
even for CC0/public-domain assets that don't legally require credit. Keeping
this current means we can never get caught flat-footed on licensing.

## Current status

Milestone 2 (Phase 1) introduced our first real art — see the Tiny Dungeon entry
below. All on-screen art (floor, wall, hero) now comes from the sprite sheet;
the local player is identified by their yellow name label rather than a drawn ring.

### Derived — depth-biome sheets (from Kenney Tiny Dungeon)
- Source URL: https://kenney.nl/assets/tiny-dungeon (base art)
- Author / creator: Kenney (base art); derivation by this project
- License: CC0 1.0 (base is public domain; modifications released under the same)
- License URL: https://creativecommons.org/publicdomain/zero/1.0/
- Used for: the depth-biome tile kits (M15). Each is a full clone of the packed
  sheet with the 19 wall/floor/void kit tiles re-derived (explicit palette
  remap + rule-placed pixel detail) and one extension row (frames 132–136) of
  anti-tiling wall variants: 132–134 = brick-face (#40) rolls, 135–136 =
  back-edge (#26) rolls. All other frames are unmodified so sprite indices
  stay identical across sheets. Shipped sheets, all under
  `client/public/assets/tiny-dungeon/`:
  - `tilemap_overgrown.png` (floors 5–9): gray-green stone + warm turf;
    hanging vines, underside moss, leaf-litter/moss floor variants.
  - `tilemap_crypt.png` (floors 10–14): bone-pale masonry + dusty umber floor;
    cracks, weathering chips, skulls set into some face variants, bone-pile
    and cracked-slab floor variants. (Added 2026-07-18.)
  - `tilemap_ember.png` (floors 15+): charcoal basalt + brown-ash floor,
    near-black void; glowing mortar cracks, stray ember pinpricks,
    ember-cluster and glowing-seam slab floor variants. (Added 2026-07-18.)
  - `tilemap_frost.png` (special floors, not in the depth bands): blue-white
    ice bricks + packed-snow floor; icicles on brick undersides, sparkle
    glints, ice-slick and cracked-ice floor variants. (Added 2026-07-18.)
  - `tilemap_goldvault.png` (special floors): dark gilded masonry + bronze
    floor; gold mortar veins, gems set into some faces, coin-pile and
    inlaid-seam floor variants. (Added 2026-07-18.)
  - `tilemap_flesh.png` (special floors): muted rose "living" masonry + dark
    meat floor; crawling veins, an eye in some faces, pustule-cluster and
    sinew floor variants. (Added 2026-07-18.)
- Modifications: as above; generated by `assets-src/biomes/build_biomes.py`
  (gitignored tooling; the palette map + detail rules there are the source of
  record until a sheet is hand-edited, at which point the PNG itself is).
- Date added: 2026-07-18

## Planned sources (Milestone 2)

The plan is to pull top-down dungeon art from **Kenney** (https://kenney.nl),
whose packs are released under **CC0 1.0** (public domain — free for any use,
including commercial, no attribution required). Likely packs:

- Kenney — "Tiny Dungeon" (16×16 top-down dungeon tiles + characters)
- Kenney — "Roguelike/RPG pack"
- Kenney — UI packs for inventory/HUD

When we add any asset, append an entry below in this format:

```
### <asset / pack name>
- Source URL:
- Author / creator:
- License: (e.g. CC0 1.0, CC-BY 4.0)
- License URL:
- Used for:
- Modifications: (none / describe)
- Date added:
```

> Note: if we ever use a **CC-BY** asset, attribution becomes *required* — the
> credit must appear both here and somewhere user-visible (e.g. a credits
> screen). CC0 assets we still log here for our own records.
